Проект: Доска задач (Task Board)

Описание:
Это простое веб-приложение для управления задачами в виде drag-and-drop доски.
Поддерживает:
- добавление задач
- редактирование и удаление задач
- сохранение в localStorage
- фильтрацию по статусу
- кастомный UI и модальное окно
- подсчет количество задач и сумма каждой зоны на знак "i"


Технологии:
- HTML
- CSS
- JavaScript (без фреймворков)
- Локальное хранилище (localStorage)

Как запустить:
1. Скачайте или клонируйте проект.
2. Откройте файл `index.html` в браузере (двойной клик или через Live Server).
3. Приложение готово к использованию.
